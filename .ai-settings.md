## Purpose

Docker-based buildchain that compiles Asterisk PBX into signed Alpine `apk`
packages for multiple Asterisk lines and multiple architectures, published as a
ready-to-use apk repository. Part of the **rtc** (real-time communications)
collection alongside `asterisk`, `asterisk-basic-agi`, `vb`.

## Knowledge base

Collection MOC: `~/notes/workspace/rtc/RTC.md`. This repo's space:
`~/notes/workspace/rtc/asterisk-alpine/` (MOC `Asterisk Alpine.md` + `Decisions/`,
`Sessions/`, `Backlog.md`). Read the collection MOC then this repo's MOC at
session start.

## Quick facts

- Alpine base **3.24**; repo layout `v3.24/main/<arch>/` + shared `noarch/`.
- Pin tag **`@andrius-asterisk`** (avoids clashing with Alpine's own asterisk).
- Live: x86_64 + aarch64 for all 8 lines; armv7 + armhf for 22 / 22-cert / 23.
- Publish is **merge-with-live** (never wipes); deploy allowed from `main` only.
